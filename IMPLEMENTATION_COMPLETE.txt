🚀 QUICK START (5 MINUTES)
═══════════════════════════════════════════════════════════════════════════

1. Setup Environment:
   $ cp .env.uima.example .env
   # Edit .env and add your OPENROUTER_API_KEY

2. Start Services:
   $ docker-compose up -d

3. Access Services:
   • Open WebUI:   http://localhost:3000
   • Adminer:      http://localhost:8081
   • API Docs:     http://localhost:3000/docs

4. Create Profile:
   $ TOKEN="your_token"
   $ curl -X POST http://localhost:3000/api/v1/user-profiles/profiles/user \
     -H "Authorization: Bearer $TOKEN" \
     -H "Content-Type: application/json" \
     -d '{"job":"Engineer","tone_preference":"technical"}'

5. Chat with Context:
   • Profile context automatically injected
   • AI responds with personalized context
